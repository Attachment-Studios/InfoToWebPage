InfoToWebPage

About

InfoToWebPage is a simple tool where you give some data and it results to a simple web-page html and css. Which you can copy for code and then with slight editing your site is ready to publish.


How It Works

In short, this tool is based on pure Javascript. HTML and CSS make it look good enough for one to understand what he needs to do. Other than that it's internal processes are easy to understand. This tool takes input from user in text. Then while comipiling it automatically adds those tags for htlm accordingly with respect to it place, where it is located and then generates CSS.
